## 二、接口测试用例

### 1. 认证接口

#### API-001: 登录接口
```
POST /api/token/
请求参数:
{
  "phone": "13800138000",
  "password": "base64_encoded_password"
}

预期响应:
{
  "access": "jwt_access_token",
  "refresh": "jwt_refresh_token",
  "user": {
    "id": 1,
    "phone": "13800138000",
    "username": "张三"
  }
}
```

#### API-002: Token 刷新接口
```
POST /api/token/refresh/
请求参数:
{
  "refresh": "jwt_refresh_token"
}

预期响应:
{
  "access": "new_jwt_access_token"
}
```
